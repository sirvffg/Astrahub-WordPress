# 更新日志

## 0.1.5 - 2026-06-24

- 修复友链管理待审核邀请在 WordPress 后台缓存旧资源时仍只显示“拒绝”的问题。
- 后台资源版本升级到 0.1.5，强制刷新 `wp-astrahub-admin.js/css` 缓存。
- 待审核邀请操作区固定显示“同意 / 拒绝”双按钮，审核弹窗文案同步为“同意友链邀请”。

## 0.1.4 - 2026-06-24

- 根治“我已关注”单向删除刷新后回弹：删除单向关注时只移除本站 -> 对方的关系边，不再走双方解除关系。
- “我已关注”删除不会再发送解除关系邮件；只有互相关注的“解除关系”才通知对方。
- 友链星球按关系状态分流：following 显示删除友链，mutual 显示解除关系。
- 保持待审核邀请的操作判定为 inbox + pending，避免“对方邀请我”时因为我已单向关注对方而缺少同意按钮。
- 修复友链管理待审核邀请操作区在窄列中只露出“拒绝”的问题，固定显示“同意 / 拒绝”双按钮。

## 0.1.3 - 2026-06-24

- 修复解除关系后只更新本地状态、未立即推送最新友链快照，导致刷新页面又从 Hub 旧关系边恢复为“我已关注”的问题。

## 0.1.2 - 2026-06-24

- 修复解除友链关系后，WordPress 本地友链可能因为 URL 尾斜杠、大小写或站点标记匹配差异没有删除，导致后台仍显示“我已关注”的问题。
- 修复“友链管理 > 对方邀请”在部分数据状态下只显示“拒绝”、不显示“通过”的问题。
- 修复“全部”筛选没有完整展示所有邀请状态的问题。
- 修复解除关系成功提示中的中文乱码。

## 0.1.1 - 2026-06-24

- 修复解除友链时解除原因为空会被编码成错误请求体，导致 Hub 返回 `invalid request body` 的问题。
- 修复实时连接令牌接口因签名请求体不一致导致 `/realtime/token` 返回 `401 Unauthorized` 的问题。
- 修复“最近同步”面板读取不到服务端已保存同步状态的问题。
- 调整“最近同步”时间展示，优先显示最近一次同步尝试的更新时间。
- 清理 GitHub 仓库中的开发文档、前端源码和本地工具文件，只保留插件运行与发布所需文件。

## 0.1.0

- 首个公开版本。
